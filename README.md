# ETF Weekly Performance Tracker

Automated weekly tracker that fetches market data for two ETFs, generates a 
combined performance chart, and delivers a formatted summary report via email.

## What it does

Pulls the last completed trading week of price data via Yahoo Finance, computes 
open-to-close weekly return for each ETF, generates a side-by-side dark-theme 
chart with annotated highs/lows, and sends the chart as an email attachment 
with a plain-text comparison table.

## Setup

Install dependencies with `pip install -r requirements.txt`. Create a `.env` 
file in the project root with your email credentials:
```
EMAIL=your@email.com
EMAIL_PSW=your_app_password
```

*Note that the EMAIL_PSW differs from the normal login password.It is rather a 16-character password randomly generated by providers like Yahoo, Gmail or Outlook and used as a credential specifically for third-party applications connecting via SMTP.

Then run with `python etf_tracker.py`. To automate weekly delivery, schedule 
it via Windows Task Scheduler or cron on Linux/macOS.

## Stack

Python · yfinance · pandas · matplotlib · smtplib · python-dotenv
